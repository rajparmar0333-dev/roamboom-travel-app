# RoamBoom – Travel Planning & Exploration App

## Overview

RoamBoom is a UI/UX-based travel planning and exploration application designed to help users research destinations, discover new places, connect with fellow travelers, and plan trips through an interactive and user-friendly interface. The application focuses on ease of use, destination discovery, social travel interaction, and visually engaging navigation.

---

## Features

* User Login & Authentication
* DigiLocker-Based Verification
* Destination Search
* Trending & Recommended Places
* Category-Based Exploration
* Wishlist System
* Explore Together Feature
* Public Group Chats
* Travel Guides & Agencies
* Profile & Settings Management
